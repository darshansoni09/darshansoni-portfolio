Vehicle Tracking & Fleet Monitoring Platform

Overview

A real-time fleet tracking platform for leased vehicles. The system processes high-frequency telemetry packets and provides live location updates, alerts and device management through a mobile dashboard.

Architecture

Vehicle Device → Telemetry Packets → Node.js Processing Layer → LRU Cache → Kafka Event Stream → WebSocket Updates → Flutter Dashboard

Engineering Work

Implemented an LRU caching layer to store frequently accessed data such as geocoding information and alert configurations, reducing database queries during high-frequency telemetry updates.

Designed Kafka-based event streaming for server-to-server communication, enabling scalable processing of telemetry events.

Implemented WebSocket communication to deliver real-time vehicle updates and sensor alerts to the mobile dashboard.

Designed an OTA firmware update system using MQTT by splitting firmware binaries into smaller chunks and delivering them sequentially to devices.
